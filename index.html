Who am I?

I’m Chatchawan L Suwaratana. I’m an aspiring web developer from Austin, Texas. I'm studying JavaScript, HTML, CSS, and JQuery at breakneck speeds. I only started weeks ago and I've already learned so much! I’m currently making small video games and web apps that I find fun. My passion is to imbue a sense of whimsicality into everything I create. I'm most interested in working with a team of experienced developers to learn best how to create. I have a lot of skills to accrue, and working with others is how I learn best. A team-centric work environment pushes me to do my best. I enjoy supporting, impressing and -sometimes- competing with teammates!

Example App

This application creates an array of homework questions. The function to run it is worded as "createHomework(operator, minNum, maxNum, numberOfProblems)" so you can see that it can create a wide range of homework problems. I'm still working on how exactly to give it those inputs through the browser, but I'll learn soon. This app was built for homeschoolers. I grew up as one myself, and I remember my mother running out of homework to give us. With an expanded set of inputs, any math problems can be procedurally generated to create infinite homework.
